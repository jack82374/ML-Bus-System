# ML Bus System

ML Bus System is a website which uses Machine Learning to provide accurate arrival time predictions for the 205 and 208 Cork city bus routes.

## Architecture

- Django
- MapLibre GL JS
- NTA GTFS Realtime Feeds
- MariaDB
- TensorFlow
- AWS Lightsail
